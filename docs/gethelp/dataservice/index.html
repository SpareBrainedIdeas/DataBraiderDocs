Data Model Service

While we are very proud how easy Data Braider is to use, it does rely on understanding the data model of Business Central.

To that end, we have a per-instance Data Model Reference Service – a per-incident way of asking us to answer your questions about where to find the data you need (for reading or writing) lives and how to configure it for Data Braider use.

Now, it may be possible that there is no easy way – some things are best still solved through Custom API’s build as an extension – if that’s the case, we’ll waive the incident.

To purchase a 10 pack of incidents, please visit our Data Model Reference purchase page.
